# Lab task for web programming

### Technologies
- Backend: Spring MVC, Spring Data
- Frontend: React, Redux, Bootstrap
- Persistence: Postgres
- REST Api
- Adaptive layout for mobile, tablet and desktop
- User registration and authorization system

### Requirements
- jdk 17
- kotlin 1.7
- maven
- docker
